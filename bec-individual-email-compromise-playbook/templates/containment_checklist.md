# Containment Checklist — IEC

## Identity
- [ ] Disable sign-in
- [ ] Revoke sessions / tokens
- [ ] Reset password
- [ ] Reset MFA / re-register
- [ ] Remove suspicious OAuth grants

## Mailbox
- [ ] Remove forwarding (including hidden)
- [ ] Remove malicious inbox rules
- [ ] Remove delegates / SendAs / FullAccess
- [ ] Review auto-replies
- [ ] Message trace: identify outbound fraud/phish
- [ ] Purge/quarantine malicious outbound messages where possible

## Business/Comms
- [ ] Notify Finance/AP if payment fraud risk
- [ ] Internal warning to recipients (if phish sent internally)
- [ ] External notification if appropriate
- [ ] Add monitoring for 7–14 days
